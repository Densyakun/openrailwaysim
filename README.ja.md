# Open Railway Sim

Open Railway Sim (ORS) はオープンソースの鉄道シミュレーターです。

- Next.js アプリ

#### シミュレーションするもの

- 粘着式鉄道
- ラック式鉄道
- 鋼索鉄道
- 鉄輪式リニア

#### シミュレーションしないもの

- モノレール
- ゴムタイヤ、案内軌条式鉄道
- 浮上式鉄道
- 索道、ロープウェイ

## Quick Start

1. Clone or fork `openrailwaysim`:

```sh
git clone https://github.com/Densyakun/openrailwaysim.git
```

2. Install all dependencies:

```sh
cd openrailwaysim
npm install
```

3. Run your App:

```sh
npm run dev
```
